Feature Scaling and Train-Test Split¶

  • Separates the features (X) and the target variable (y) for model training.

  • Applies Standard Scaling to normalize the features — useful for algorithms that are sensitive to feature scales like logistic regression or SVM.

  • Splits the dataset into training (80%) and testing (20%) sets to evaluate model performance effectively. A fixed random_state ensures reproducibility.

  • y_pred: Predictions made by the model on unseen test data.
  • accuracy_score: Measures how many predictions were correct.
  • confusion_matrix: Shows counts of true/false positives/negatives.
  • classification_report: Gives precision, recall, and F1-score per class.

Conclusion¶

In this project, we built a Logistic Regression model to predict survival outcomes of passengers aboard the Titanic. After preprocessing the dataset—handling missing values, encoding categorical features like 'Sex' and 'Embarked', and scaling the numerical features—we split the data into training and testing sets. The final model achieved an accuracy of approximately 79.3% on the test set. The classification report indicated balanced performance with an F1-score of 0.83 for non-survivors and 0.77 for survivors. Overall, the model effectively captured key survival patterns and provides a solid baseline for future improvements.
